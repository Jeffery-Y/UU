# UU
基于Bmob后端云的即时通讯系统（仿qq界面），包括以下四大核心功能模块，包括：
1. 完善的用户管理功能模块，具体包括：
 * 用户注册（支持验证手机/账户密码注册）
 * 登录（支持用户名/手机/邮箱和密码登录、验证手机一键登录）
 * 修改密码（支持验证手机/旧密码修改）
 * 绑定手机及邮箱
 * 更新地理位置信息
 * 同一账号多处登陆强制下线
 * 消息通知的管理（声音和震动）
 * 修改其他个人信息（头像/昵称/性别）
2. 丰富的好友管理功能模块，具体包括：
 * 添加好友
 * 删除好友
 * 附近的人
 * 黑名单管理
 * 获取好友列表
 * 浏览好友（支持查找和按首字母查看）
 * 查找用户（支持精确查找和模糊查找）
 * 扫描码添加好友（支持二维码和条形码的生成与扫描）
3. 稳定的即时通讯功能模块，具体包括：
 * 支持定时检测服务
 * 支持消息重发机制
 * 支持陌生人间聊天
 * 支持消息回执（已发送/已阅读）
 * 支持文本/小表情/图片/音频/地理位置等多种格式的富媒体消息的发送和接收
4. 强大的聊天会话功能模块，具体包括：
 * 支持文字消息朗读
 * 支持语音转换为文字
 * 语音消息的录制和播放
 * 地理位置的发送和查看
 * 支持小表情/本地图片/拍摄图片的查看
